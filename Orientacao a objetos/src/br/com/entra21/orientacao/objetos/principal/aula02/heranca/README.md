# Aprendendo Herança

## Herança é a abstração para aproveitar atributos e metodos de uma outra classe

> Essa classe que está provendo atributos e metodos podemos chama-la de super classe
> Para realizar essa herança em uma sub classe temos que usar a palavra reservado **extends**
> Quando uma subclasse herda atributos da super calsse não é mais necessario manter meus atributos e metodos escritos na minha sub classe.
